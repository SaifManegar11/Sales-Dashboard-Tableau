# Sales-Customer-Dasboard

This project is an interactive Sales and Customer Analytics Dashboard created using Tableau. The dashboard provides insights into sales performance, profit trends, customer behavior, and order analysis for the year 2023 through visually interactive charts and KPIs.

The project contains two dashboards: a Sales Dashboard and a Customer Dashboard. The Sales Dashboard shows key metrics such as sales, profit, quantity, monthly trends, and product subcategory performance, while the Customer Dashboard analyzes customer count, orders, sales per customer, and top customers by profit. These dashboards help in tracking business performance and making data-driven decisions through interactive visualizations.
